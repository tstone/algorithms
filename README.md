My tinkerings and implementations around Algorithms 4th edition.

## Scala & Java

To run algorithms:

```shell
$ cd jvm

```

## Javascript

Requirements:

  1.  Install Node.js
  2.  Install jasmine-node `$ sudo npm install jasmine-node -g`

To run algorithms:

```shell
$ cd js
$ jasmine-node spec/
```
